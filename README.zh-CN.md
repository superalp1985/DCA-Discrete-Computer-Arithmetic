# DCA：离散计算机算术

这是一个面向有限计算的“离散计算机算术”（DCA, Discrete Computer Arithmetic）整理稿，提供中文和英文两个版本。

本文不是要替代现有数学，也不是要宣称一个更大的数学体系。它只是围绕能被有限计算表示、执行、检查或近似的数学内容，整理定义、论证过程、实现边界和已有工作实例。

## 范围

整理稿保留 43 个部分，覆盖算术、代数、离散分析、几何、逻辑、NTT、概率、拓扑、有限域代数几何、形式化验证、量化 AI 计算、元胞自动机、定理证明自动化，以及相关工程实例。

## 这是什么

- 一个面向有限计算的写作框架。
- 一个供程序、芯片、证明助手和有限数据结构参考的草稿。
- 一组定义、证明草图、实现说明和参考资料。

## 这不是什么

- 不是对经典数学的替代。
- 不是对连续数学的否定。
- 不是物理理论。
