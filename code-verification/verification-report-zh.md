## 第4章：离散几何

### 章节概述

第4章涵盖了离散几何的基本概念，包括：
- 离散点和向量
- 距离度量（曼哈顿距离、汉明距离、图距离）
- 离散线和圆
- 多边形操作
- 几何相交测试
- 基于网格的算法

### 实现详情

#### 1. 基础几何原语

**Point 类**
- 表示 Z^d 空间中的离散点
- 支持向量加法、减法和标量乘法
- 实现曼哈顿距离、切比雪夫距离和平方欧几里得距离
- 添加了 `__lt__` 方法以支持堆操作

**DiscreteLine 类**
- 由起点和方向向量表示的离散线
- 支持参数化点生成
- 点共线检测

**DiscreteCircle 类**
- 支持曼哈顿度量和平方欧几里得度量
- 边界点和内部点枚举
- 点包含检测

#### 2. 距离度量

**HammingDistance 类**
- 整数和字节序列的汉明距离计算
- 度量公理验证

**GraphDistance 类**
- 使用 BFS 计算图距离
- 支持单对最短路和全源最短路

#### 3. 多边形操作

**Polygon 类**
- 周长计算（使用曼哈顿距离）
- 面积计算（使用鞋带公式）
- 点在多边形内检测（射线投射算法）
- 包围盒计算

#### 4. 网格算法

**Grid 类**
- 2D 离散网格表示
- 障碍物管理
- BFS 最短路径算法
- A* 最短路径算法（曼哈顿启发式）

#### 5. 相交测试

**Intersections 类**
- 线段相交检测（使用方向和点测试）
- 点在圆内检测

## 第5章：逻辑与推理

### 章节概述

第5章涵盖了逻辑与推理的核心概念，包括：
- 命题逻辑和布尔运算
- 真值表生成
- SAT求解器基础（DPLL算法）
- 逻辑推理规则
- 合取范式(CNF)转换
- 有效性和可满足性检查

### 实现详情

#### 1. 基本逻辑运算

**LogicalConnective 枚举**
- AND, OR, NOT, IMPLIES, IFF, XOR

**Literal 类**
- 表示文字（变量或其否定）
- 支持求反操作
- 在赋值下求值

**Clause 类**
- 子句是文字的析取
- 单元子句检测
- 在赋值下简化和求值
- 空子句检测（矛盾）

**CNFFormula 类**
- 合取范式公式
- 子句的合取
- 变量集管理

#### 2. 真值表生成

**TruthTable 类**
- 为任意布尔函数生成真值表
- 重言式检测
- 矛盾式检测
- 可满足性检测
- 美化输出

#### 3. SAT求解器 (DPLL算法)

**DPLLSolver 类**
- 完整的DPLL实现
- 单元传播
- 纯文字消除
- 变量选择启发式
- 回溯机制
- 决策和回溯计数

#### 4. 逻辑推理规则

**InferenceRules 类**
- 假言推理 (Modus Ponens)
- 拒取式 (Modus Tollens)
- 假言三段论 (Hypothetical Syllogism)
- 析取三段论 (Disjunctive Syllogism)
- 归结规则 (Resolution)

#### 5. 公式转换和等价性

**FormulaConverter 类**
- 蕴涵转换为AND/OR: p → q ≡ ¬p ∨ q
- 异或转换: p ⊕ q ≡ (p ∧ ¬q) ∨ (¬p ∧ q)
- 当且仅当转换: p ↔ q ≡ (p → q) ∧ (q → p)
- 德摩根定律验证
- 分配律验证

#### 6. 命题逻辑求值器

**PropositionalLogic 类**
- 表达式求值
- 反例查找

#### 7. 证明系统

**SimpleProof 类**
- 真值表证明
- 推理验证
- 前提到结论的验证
