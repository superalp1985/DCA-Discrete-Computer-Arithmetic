## 实现细节

### 核心数据结构
- **Graph**: 有限图表示
- **Laplacian**: 图拉普拉斯矩阵 L = D - A

### 关键算法实现

1. **图拉普拉斯计算**
   ```python
   def laplacian_matrix(self) -> List[List[int]]:
       D = np.array(self.degree_matrix())
       A = np.array(self.adjacency_matrix())
       L = D - A
       return L.tolist()
   ```

2. **二次型验证**
   ```python
   def verify_quadratic_form(self):
       x^T L x = sum_{(u,v) in E} (x_u - x_v)^2
   ```

3. **连通分量**
   - BFS遍历
   - 分量计数

4. **谱性质**
   - 特征值计算
   - 零空间维度

### 详细测试结果

1. **拉普拉斯定义验证**
   - 三角形图: L = [[2,-1,-1],[-1,2,-1],[-1,-1,2]]
   - 每个元素满足 L[i,j] = D[i,j] - A[i,j]

2. **二次型恒等式验证**
   - 路径图测试
   - 多个向量验证
   - 精度 < 1e-6

3. **拉普拉斯性质验证**
   - 对称性: L[i,j] = L[j,i]
   - 行和为零: Σ_j L[i,j] = 0
   - 正定: 所有特征值 ≥ 0

4. **零空间维度验证**
   - 连通图: null_dim = 1
   - 断开图: null_dim = components
   - 常向量是基

5. **谱性质验证**
   - 0总是特征值
   - 第二特征值 > 0 iff 连通
   - 所有特征值非负
