### 关键算法实现

1. **Bellman递推**
   ```python
   def bellman_backup(self, V_next: Dict[State, float], s: State) -> float:
       min_value = float('inf')
       for a in self.actions:
           next_state = self.transition(s, a)
           value = self.cost(s, a) + V_next[next_state]
           if value < min_value:
               min_value = value
       return min_value
   ```

2. **反向归纳**
   - 从终端时刻T开始，向前计算每个时刻的最优值函数
   - 保证在有限时间内完成计算

3. **无限时域折扣**
