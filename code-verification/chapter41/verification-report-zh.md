## 一、章节概述

第41章《离散物理映射》探讨了如何将物理模型转化为离散的、可计算的形式。本章的核心思想是将连续的物理定律转化为离散的有限状态系统，使其能够在计算机上进行精确的计算和验证。

### 主要内容

1. **离散作用量原理**：将连续作用量积分转化为离散求和 `S[path] = Σ_t L(x_t, x_{t+1})`
2. **离散欧拉-拉格朗日方程**：通过有限差分推导运动方程
3. **离散诺特定理**：证明守恒量在离散对称变换下的不变性
4. **格点场模型**：在离散空间上定义场论
5. **粒子系统**：离散多体系统的数值模拟
6. **有限差分方程**：离散微分方程的数值求解

## 二、实现细节

### 2.1 核心数据结构

```python
class DiscreteAction:
    """离散作用量函数"""
    def compute(self, path: List[float]) -> float:
        return sum(self.lagrangian(path[t], path[t+1])
                   for t in range(len(path) - 1))
```
