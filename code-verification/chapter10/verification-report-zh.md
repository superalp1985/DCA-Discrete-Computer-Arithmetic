## 一、章节概述

第10章《离散复分析与二元数》主要讨论：

1. **有限域扩张**：在有限结构中模拟复数，使用 `F_p[i] = F_p[x]/(x^2 + 1)`
2. **二元数（对偶数）**：定义为 `a + bε`，满足 `ε^2 = 0`，用于自动微分
3. **自动微分**：前向模式自动微分的代数骨架

核心公式：
- 有限域扩张：`F_p[i] = F_p[x]/(x^2 + 1)`
- 二元数乘法：`(a + bε)(c + dε) = ac + (ad + bc)ε`
- 自动微分：`f(a + bε) = f(a) + b f'(a) ε`
