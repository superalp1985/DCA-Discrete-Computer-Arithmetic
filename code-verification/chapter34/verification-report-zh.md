## 实现细节

### 核心数据结构
- **FiniteField**: 有限域F_q表示
- **LinearCode**: 线性码，由生成矩阵G或校验矩阵H定义
- **SyndromeDecoder**: 基于综合表的译码器

### 关键算法实现

1. **线性码编码**
   ```python
   def encode(self, message: List[int]) -> List[int]:
       m = np.array(message)
       c = np.dot(m, self.G) % self.q
       return c.tolist()
   ```

2. **综合计算**
   ```python
   def syndrome(self, received: List[int], H: List[List[int]]) -> List[int]:
       r = np.array(received)
       H_mat = np.array(H)
       s = np.dot(H_mat, r) % self.q
       return s.tolist()
   ```

3. **综合译码器**
   - 构建综合到错误模式的查找表
   - 支持最多max_error_weight个错误的纠正

### 复杂度分析

- 编码复杂度: O(k × n)
- 综合计算复杂度: O((n-k) × n)
- 译码复杂度: O(q^t) for lookup table construction
