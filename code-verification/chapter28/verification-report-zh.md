## 四、NP验证验证

### 4.1 验证原理

NP的定义：存在多项式长度的证书，可以在多项式时间内验证。

```
L ∈ NP ⇔ ∃ polynomial-time verifier V:
∀x ∈ L, ∃certificate c: V(x, c) = True
∀x ∉ L, ∀certificate c: V(x, c) = False
```

### 4.2 实现代码

```python
def verify_sat(self, formula: str, assignment: Dict[str, bool]) -> bool:
    clauses = []
    for clause_str in formula.split(' and '):
        clause_str = clause_str.strip('()')
        literals = clause_str.split(' or ')
        clause = []
        for lit in literals:
            lit = lit.strip()
            if lit.startswith('not '):
                var = lit[4:]
                clause.append((var, False))
            else:
                clause.append((lit, True))
        clauses.append(clause)

    for clause in clauses:
        satisfied = False
        for var, value in clause:
            if var in assignment and assignment[var] == value:
                satisfied = True
                break
        if not satisfied:
            return False
    return True
```

### 4.3 验证测试

- **SAT验证测试**：验证可满足公式的赋值
- **SAT否定测试**：验证不可满足赋值的拒绝
- **子集和验证测试**：验证子集和的证书
- **图着色验证测试**：验证2-可分图的着色

### 4.4 验证结果

|---------|---------|------|------|
| SAT验证 | 1 | 1 | 0 |

| 子集和验证 | 1 | 1 | 0 |
| 图着色验证 | 1 | 1 | 0 |

**结论**：NP问题的证书验证可以在多项式时间内完成。

## 七、多项式时间归约验证

### 7.1 验证原理

多项式时间归约：

```
A ≤p B ⇔ ∃ polynomial-time f: x → f(x) such that
x ∈ A ⇔ f(x) ∈ B
```

### 7.2 验证测试

- **4-SAT到3-SAT归约**：验证任何4-SAT实例可以转换为等价的3-SAT实例
