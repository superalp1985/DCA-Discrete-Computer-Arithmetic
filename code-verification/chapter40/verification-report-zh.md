## 实现细节

### 核心数据结构
- **Opcode**: 字节码操作码枚举
- **Instruction**: 单条指令表示
- **Program**: 程序(指令列表+常量)
- **InterpreterState**: 解释器执行状态
- **BootstrapInterpreter**: 自举解释器

### 关键算法实现

1. **程序编码**
   ```python
   def encode(self) -> str:
       inst_str = "|".join(inst.encode() for inst in self.instructions)
       const_str = ",".join(map(str, self.constants))
       return f"I:{inst_str}|C:{const_str}"
   ```

2. **单步执行**
   ```python
   def step(self) -> bool:
       if self.state.fuel <= 0:
           self.state.status = "out_of_fuel"
           return False
       # Fetch and execute instruction
       inst = self.program.instructions[self.state.pc]
       self.execute(inst)
       self.state.fuel -= 1
       return True
   ```

3. **算术运算**
   - LOAD_CONST: 加载常量
   - ADD/SUB/MUL/DIV: 栈运算
   - STORE_VAR/LOAD_VAR: 变量存储

4. **控制流**
   - JMP: 无条件跳转
   - JZ: 条件跳转
   - HALT: 停止
