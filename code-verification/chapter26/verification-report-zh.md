## 三、量子门组合验证

### 3.1 验证原理

门组合的可逆性：

```
(UV)^-1 = V^-1 U^-1
```

特别地，对于自逆门（如X, SWAP, CNOT）：

```
U^2 = I => U^-1 = U
```

### 3.2 实现代码

```python
def apply_pauli_x(self, qubit: int):
    n = len(self.state)
    new_state = [0] * n
    bit_pos = qubit

    for i in range(n):
        new_i = i ^ (1 << bit_pos)
        new_state[new_i] = (new_state[new_i] + self.state[i]) % self.q

    self.state = new_state
```

### 3.3 验证测试

- **X门自逆测试**：X(X(|0>)) = |0>
- **SWAP门自逆测试**：SWAP(SWAP(|ab>)) = |ab>

### 3.4 验证结果

| 测试类型 | 测试数量 | 通过 | 失败 |
|---------|---------|------|------|
| X门组合 | 1 | 1 | 0 |
| SWAP门组合 | 1 | 1 | 0 |

**结论**：量子门组合的可逆性实现正确。

## 六、CNOT逻辑验证

### 6.1 验证原理

CNOT（受控非）门的行为：

```
CNOT(|c,t>) = |c, t⊕c>
```

其中c是控制比特，t是目标比特，⊕是异或。

### 6.2 实现代码

```python
def apply_cnot(self, control: int, target: int):
    n = len(self.state)
    new_state = [0] * n
    control_pos = control
    target_pos = target

    for i in range(n):
        control_bit = (i >> control_pos) & 1
        if control_bit:
            target_bit = (i >> target_pos) & 1
            if target_bit == 0:
                new_i = i | (1 << target_pos)
            else:
                new_i = i & ~(1 << target_pos)
            new_state[new_i] = (new_state[new_i] + self.state[i]) % self.q
        else:
            new_state[i] = (new_state[i] + self.state[i]) % self.q

    self.state = new_state
```

### 6.3 验证测试

| 输入 | 控制比特 | 目标比特 | 期望输出 |
|------|---------|---------|---------|
| \|11> | 1 | 1 | \|01> |
| \|00> | 0 | 0 | \|00> |
| \|10> | 1 | 0 | \|10> |
| \|01> | 0 | 1 | \|11> |

### 6.4 验证结果

| 测试类型 | 测试数量 | 通过 | 失败 |
|---------|---------|------|------|
| CNOT逻辑 | 4 | 4 | 0 |

**结论**：CNOT门的逻辑实现正确，满足受控非的预期行为。
