# 第二十章验证报告：离散随机过程与鞅

## 章节概述

本章验证了DCA系列第二十章的核心概念：**离散随机过程与鞅**。随机过程可以写成有限时间、有限状态、有限概率权重的模型。

## 实现详情

### 1. 有限概率空间
- 实现了有限概率空间`FiniteProbabilitySpace`
- 支持事件概率计算
- 验证了总概率为1

### 2. 离散时间随机过程
- 实现了离散时间随机过程`StochasticProcess`
- 支持期望和方差计算
- 实现了路径模拟

### 3. 过滤与条件期望
- 实现了过滤表示信息随时间的演化
- 实现了条件期望计算
- 验证了可测性

### 4. 鞅性质验证
- 实现了鞅性质检查：`E[X_{t+1}|F_t] = X_t`
- 实现了上鞅和下鞅检查
- 验证了对称随机游走的鞅性质

### 5. 停时
- 实现了停时验证
- 验证了停时的可测性
- 实现了可选停定理验证

### 6. 马尔可夫链
- 实现了有限状态马尔可夫链
- 支持n步转移概率计算
- 验证了转移矩阵性质

### 7. 二项随机游走
- 实现了二项随机游走过程
- 验证了对称游走的鞅性质
- 验证了非对称游走不是鞅
