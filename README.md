# DCA: Discrete Computer Arithmetic

A modest finite-computation-oriented draft of Discrete Computer Arithmetic (DCA), with Chinese and English versions.

This repository is not claiming to replace existing mathematics. It organizes definitions, arguments, implementation boundaries, and existing work around the parts of mathematics that can be represented, executed, checked, or approximated by finite computation.

## Scope

The draft keeps 43 sections, covering arithmetic, algebra, discrete analysis, geometry, logic, NTT, probability, topology, finite-field algebraic geometry, formal verification, quantized AI computation, cellular automata, theorem proving, and related implementation examples.

## What This Is

- A finite-computation-oriented writing framework.
- A reference draft for programs, chips, proof assistants, and finite data structures.
- A collection of definitions, proof sketches, implementation notes, and references.

## What This Is Not

- It is not a replacement for classical mathematics.
- It is not a claim that continuous mathematics is invalid.
- It is not a physical theory.
